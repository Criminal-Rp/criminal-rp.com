# criminal-rp.com
Criminal RP Mobile — современный CRMP-проект с открытым миром и качественным RolePlay. Начни путь с нуля, зарабатывай деньги, покупай автомобили, дома и бизнесы, вступай во фракции или покоряй криминальный мир. Регулярные обновления, уникальные системы, активное сообщество и море возможностей ждут именно тебя!
